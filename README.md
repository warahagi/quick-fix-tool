# 🚀 QuickSnippets

ブラウザで動く、シンプルで強力なスニペット管理ツールです。  
日々のコーディング、定型文、IT用語のメモなどを効率的に管理し、素早くアクセスできます。

![Main View](https://img.shields.io/badge/Status-Public-success)
![License](https://img.shields.io/badge/License-MIT-blue)

## ✨ 主な機能

- **クイック検索**: タイトルや内容から瞬時にスニペットを見つけ出します。
- **カテゴリー分類**: 学習用、SQL、JSなど、自由にカテゴリーを分けて管理。
- **学習モード (Flashcards)**: 登録したスニペットをカード形式で学習し、知識の定着を図れます。
- **ファイル連携 (File System Access API)**: ブラウザからローカルの `snippets.json` を直接読み書き可能。（※対応ブラウザのみ）
- **CSV インポート/エクスポート**: Excelなどで編集したデータを一括で取り込めます。
- **ダークモード対応**: OSの設定に合わせて自動でテーマが切り替わります。

## 🛠 使い方

1. `index.html` をブラウザで開きます。
2. 右下の「＋」ボタンから新しいスニペットを追加します。
3. カテゴリータブを切り替えてスニペットを閲覧します。
4. 「学習モード」をクリックして、登録内容を復習できます。

### おすすめ設定
「連携」ボタンを使用して、プロジェクト内の `snippets.json` と連携させると、保存時にローカルのファイルが直接更新されるため、非常に便利です。

## 📂 ファイル構成

- `index.html`: メインアプリケーション
- `flashcard.html`: 学習モード画面
- `snippets.json`: データ保存ファイル（JSON形式）
